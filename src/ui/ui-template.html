导出 LaTeX/Beamer
将整个 Deck 导出为可编译的 Beamer .tex 文件。公式、图表、定理、表格、参考文献、交叉引用均自动转换。
导出后用 pdflatex + bibtex 编译：
pdflatex slides.tex && bibtex slides && pdflatex slides.tex && pdflatex slides.tex
